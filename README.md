# Keskustelusovellus

Sovelluksessa käyttäjä voi tunnuksen luomisen jälkeen luoda uusia keskusteluaiheita tai lähettää viestejä vanhoihin aiheisiin. Käyttäjät ovat joko peruskäyttäjiä tai moderaattoreita. 

Sovelluksen ominaisuuksia ovat:

* Etusivulta näkyy lista aiheista esim. yleistä keskustelua, myynnit ja ostot yms. Jokaisessa aiheessa näkyy myös montako keskustelaihetta ja viestiä on luotu.
* Kun keskusteluaihe on avattu, näkyy alkuperäinen viesti, aika kun se on luotu, lähettäjä ja käyttäjien lisäämät viestit. 
* Käyttäjä voi luoda uuden tilin ja kirjautua joko sisään tai ulos.
* Käyttäjä voi luoda uuden keskusteluaiheen tai lähettää viestin olemassa olevaan keskusteluaiheeseen.
* Käyttäjä voi poistaa omia viestejä ja keskusteluaiheita.
* Viesteissä näkyy viestin lisäksi lähettäjä ja kellonaika jolloin viesti on lähetetty.
* Moderaattori voi poistaa viestejä, keskusteluaiheita ja bannata käyttäjiä.
